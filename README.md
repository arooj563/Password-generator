# 🔐 Password Generator

A simple, fast, and fully customizable **Password Generator** built with **HTML, CSS, and JavaScript**.  
This tool helps users create strong, secure passwords with adjustable settings such as symbols, numbers, uppercase & lowercase letters.

---

## 🚀 Features

- Generate **strong & random** passwords  
- Choose password **length**
- Include:
  - ✔ Uppercase letters  
  - ✔ Lowercase letters  
  - ✔ Numbers  
  - ✔ Symbols  
- One-click **Copy to Clipboard**
- Clean & responsive UI  
- Beginner-friendly JavaScript logic
